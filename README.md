* Deployment instructions

1. Clone the repo
2. Run `gem install bundler` (if you don't already have it installed)
3. Run `bundle install`
4. Run `rake db:create`
5. Run `rake db:migrate`
6. Run `bin/rails s -b 0.0.0.0`
